Estruturas de Repetição em JavaScript
Sobre o Projeto

Este repositório contém a resolução de uma lista de exercícios sobre estruturas de repetição utilizando JavaScript.

A atividade tem como objetivo praticar o uso de laços de repetição para realizar contagens crescentes e decrescentes, trabalhar com incrementos personalizados e desenvolver algoritmos capazes de lidar com diferentes entradas fornecidas pelo usuário.
===========================================================================================================================================================================================================================================================
Conteúdo Desenvolvido
Exercício 1 =========================================================================================================================================================

Programa que exibe uma contagem de 6 até 12 e, ao final, mostra a mensagem "FIM".

Exercício 2 =========================================================================================================================================================

Programa que realiza uma contagem regressiva de 10 até 1 e exibe a mensagem "FIM" ao final.

Exercício 3 =========================================================================================================================================================

Programa que exibe uma contagem de 0 até 30, avançando de 3 em 3.

Exercício 4 =========================================================================================================================================================

Programa que realiza uma contagem regressiva de 50 até 0, diminuindo de 5 em 5.

Exercício 5 =========================================================================================================================================================

Programa que solicita ao usuário um número inteiro positivo e realiza uma contagem de 1 até o valor informado.

Exercício 6 =========================================================================================================================================================

Programa que exibe uma contagem regressiva de 30 até 1, destacando os números divisíveis por 4 entre colchetes.

Exemplo:

30 29 [28] 27 26 25 [24] ...

Exercício 7 =========================================================================================================================================================

Programa que solicita um valor inicial, um valor final e um incremento, exibindo todos os valores do intervalo informado.

O algoritmo foi desenvolvido para funcionar corretamente tanto em contagens crescentes quanto em contagens decrescentes.
